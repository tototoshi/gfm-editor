## Keybind

|Key|Action|
|---|---|
|Esc|Focus on the search box|
|Enter|Focus on textarea|
|Delete|Delete a note|
|↓|Select next note|
|↑|Select previous note|
|⌘ + →|Hide preview|
|⌘ + ←|Show preview|
